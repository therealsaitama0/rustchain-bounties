# Beacon Atlas City Visualization

Static browser visualization for bounty #159.

## Features

- Interactive canvas with city boundaries for Genesis, Aurora, and Nexus.
- Agent nodes sized by property valuation.
- Property markers around each agent with type-specific colors.
- Connection rendering for heartbeat, contract, and mayday links.
- Mayday pulse animation for agents requiring attention.
- Filters for city, role, property type, and search.
- Wheel zoom, drag pan, hover tooltips, and click-to-inspect details.

## Run

Open `index.html` in a browser. No build step or backend is required.
